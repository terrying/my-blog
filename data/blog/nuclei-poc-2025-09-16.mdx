### GiveWP Donation Plugin &lt;&#x3D; 3.16.1 - Unauthenticated PHP Object Injection

- **漏洞ID**: `CVE-2024-8353`
- **严重程度**: 🔴 CRITICAL
- **风险等级**: 极高风险 (5/5)
- **影响资产**: WordPress 站点
- **预估影响**: 数千万个

**描述**: The GiveWP – Donation Plugin and Fundraising Platform plugin for WordPress is vulnerable to PHP Object Injection in all versions up to, and including, 3.16.1. This is due to insufficient input validation on user-supplied data. An unauthenticated attacker can inject a serialized PHP object, which may allow them to execute arbitrary PHP code, depending on the presence of a suitable POP chain on the target system. This vulnerability could lead to full site compromise.


**攻击向量**: 网络扫描

**CVE编号**: CVE-2024-8353

**参考链接**: 
- [参考资料](https://nvd.nist.gov/vuln/detail/CVE-2024-8353)
- [参考资料](https://www.wordfence.com/threat-intel/vulnerabilities/wordpress-plugins/give/givewp-donation-plugin-and-fundraising-platform-3161-unauthenticated-php-object-injection)

### MCP Inspector &lt; 0.14.0 UnauthenticatedRemote Code Execution

- **漏洞ID**: `CVE-2025-49596`
- **严重程度**: 🔴 CRITICAL
- **风险等级**: 极高风险 (5/5)
