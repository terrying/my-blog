### XWiki REST API Query - SQL Injection

- **漏洞ID**: `CVE-2025-32969`
- **严重程度**: 🔴 CRITICAL
- **风险等级**: 极高风险 (5/5)
- **影响资产**: 未知
- **预估影响**: 数千个

**描述**: A SQL injection vulnerability exists in XWiki&#x27;s REST API query endpoint. An unauthenticated attacker can execute arbitrary SQL queries through the &#x27;q&#x27; parameter by manipulating the HQL query, potentially leading to data exfiltration or system compromise.


**攻击向量**: 网络扫描

**CVE编号**: CVE-2025-32969

### XWiki XML View - Sensitive Information Exposure

- **漏洞ID**: `CVE-2025-54125`
- **严重程度**: 🟠 HIGH
- **风险等级**: 高风险 (4/5)
- **影响资产**: 未知
- **预估影响**: 数千个

**描述**: A vulnerability in XWiki&#x27;s XML view functionality exposes sensitive information such as passwords and email addresses that are stored in custom fields not explicitly named as password or email. This information disclosure occurs when accessing user profiles with the xml.vm template.
