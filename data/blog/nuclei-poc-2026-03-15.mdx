- **漏洞ID**: `CVE-2025-1550`
- **CVE**: `CVE-2025-1550` (2025)
- **严重程度**: 🔴 CRITICAL
- **风险等级**: 极高风险 (5/5)
- **影响资产**: 未知
- **预估影响**: 数千个
- **EPSS**: 0.04785 (percentile 0.89321) @ 2026-03-14

**描述**: The Keras Model.load_model function permits arbitrary code execution, even with safe_mode&#x3D;True, through a manually constructed, malicious .keras archive. By altering the config.json file within the archive, an attacker can specify arbitrary Python modules and functions, along with their arguments, to be loaded and executed during model loading
