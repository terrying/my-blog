- **预估影响**: 数千个
- **EPSS**: 0.01169 (percentile 0.63284) @ 2026-06-22

**描述**: A flaw was found in the Samba printing subsystem. Samba passes the client-controlled job description string to the command configured with the &quot;print command&quot; setting via the &quot;%J&quot; substitution character without escaping shell meta characters. - **EPSS**: 0.19506 (percentile 0.97024) @ 2026-06-22
