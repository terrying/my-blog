### WP Directory Kit &lt;&#x3D; 1.4.4 - Authentication Bypass

- **漏洞ID**: `CVE-2025-13390`
- **CVE**: `CVE-2025-13390` (2025)
- **严重程度**: 🔴 CRITICAL
- **风险等级**: 极高风险 (5/5)
- **影响资产**: WordPress 站点
- **预估影响**: 数千万个
- **EPSS**: 0.0055 (percentile 0.67988) @ 2026-04-20

**描述**: The WP Directory Kit plugin for WordPress version 1.4.4 and below contains an authentication bypass vulnerability in its auto-login functionality. The vulnerability allows unauthenticated attackers to gain administrative access by exploiting a cryptographically weak token generation mechanism that uses only the first 10 characters of MD5(user_id). For user_id&#x3D;1 (typically admin), the token is always predictable.


**攻击向量**: 网络扫描

### GitLab - SAML Authentication Bypass

- **漏洞ID**: `CVE-2025-25291`
- **CVE**: `CVE-2025-25291` (2025)
- **严重程度**: 🔴 CRITICAL
- **风险等级**: 极高风险 (5/5)
- **影响资产**: GitLab 平台
- **预估影响**: 数千个
- **EPSS**: 0.20843 (percentile 0.95627) @ 2026-04-20

**描述**: ruby-saml provides security assertion markup language (SAML) single sign-on (SSO) for Ruby. An authentication bypass vulnerability was found in ruby-saml prior to versions 1.12.4 and 1.18.0 due to a parser differential. ReXML and Nokogiri parse XML differently; the parsers can generate entirely different document structures from the same XML input. That allows an attacker to be able to execute a Signature Wrapping attack. This issue may lead to authentication bypass. Versions 1.12.4 and 1.18.0 fix the issue.


**攻击向量**: 网络扫描

**CVE编号**: CVE-2025-25291
