### NotificationX Dropshipping &lt; 4.4 - SQL Injection

- **漏洞ID**: `CVE-2022-3481`
- **严重程度**: 🔴 CRITICAL
- **风险等级**: 极高风险 (5/5)
- **影响资产**: WordPress 站点
- **预估影响**: 数千万个

**描述**: The plugin does not properly sanitise and escape a parameter before using it in a SQL statement via a REST endpoint available to unauthenticated users, leading to a SQL injection


**攻击向量**: 网络扫描

**CVE编号**: CVE-2018-25031

### Apache StreamPipes &lt;&#x3D; 0.93.0 - Use of Cryptographically Weak PRNG in Recovery Token Generation

- **漏洞ID**: `CVE-2024-29868`
- **严重程度**: 🔴 CRITICAL
- **风险等级**: 极高风险 (5/5)
- **影响资产**: Apache 服务器
- **预估影响**: 数百万台

**描述**: Apache StreamPipes from version 0.69.0 through 0.93.0 uses a cryptographically weak Pseudo-Random Number Generator (PRNG) in the recovery token generation mechanism. Given a valid token it&#x27;s possible to predict all past and future generated tokens.


**攻击向量**: 网络扫描

**CVE编号**: CVE-2024-29868

**参考链接**: 
- [参考资料](https://labs.yarix.com/2024/06/cve-2024-29868)

### MCP Inspector &lt; 0.14.0 UnauthenticatedRemote Code Execution

- **漏洞ID**: `CVE-2025-49596`

- **风险等级**: 极高风险 (5/5)
