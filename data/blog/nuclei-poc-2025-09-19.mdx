### Apache HugeGraph-Server &lt;1.5.0 - Authentication Bypass

- **严重程度**: 🔴 CRITICAL

### Cobbler &#x27;XML-RPC&#x27; - Authentication Bypass

- **漏洞ID**: `CVE-2024-47533`
- **严重程度**: 🔴 CRITICAL
- **风险等级**: 极高风险 (5/5)
- **影响资产**: 未知
- **预估影响**: 数千个

**描述**: Cobbler, a Linux installation server that allows for rapid setup of network installation environments, has an improper authentication vulnerability starting in version 3.0.0 and prior to versions 3.2.3 and 3.3.7. `utils.get_shared_secret()` always returns `-1`, which allows anyone to connect to cobbler XML-RPC as user `&#x27;&#x27;` password `-1` and make any changes. This gives anyone with network access to a cobbler server full control of the server. Versions 3.2.3 and 3.3.7 fix the issue.


**攻击向量**: 网络扫描

**CVE编号**: CVE-2024-47533

**参考链接**: 
- [参考资料](https://github.com/cobbler/cobbler/commit/32c5cada013dc8daa7320a8eda9932c2814742b0)
