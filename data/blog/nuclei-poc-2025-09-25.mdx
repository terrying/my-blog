### Better Search Replace &lt; 1.4.5 - PHP Object Injection

- **漏洞ID**: `CVE-2023-6933`
- **严重程度**: 🔴 CRITICAL
- **风险等级**: 极高风险 (5/5)
- **影响资产**: WordPress 站点
- **预估影响**: 数千万个

**描述**: The Better Search Replace plugin for WordPress is vulnerable to PHP Object Injection in all versions up to, and including, 1.4.4 via deserialization of untrusted input. This makes it possible for unauthenticated attackers to inject a PHP Object. No POP chain is present in the vulnerable plugin. If a POP chain is present via an additional plugin or theme installed on the target system, it could allow the attacker to delete arbitrary files, retrieve sensitive data, or execute code.


**攻击向量**: 网络扫描

**CVE编号**: CVE-2023-6933

**参考链接**: 
- [参考资料](https://posimyth.ticksy.com/ticket/2713734/)
