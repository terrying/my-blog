### Ultimate Member &lt; 2.1.12 - Unauthenticated Privilege Escalation via User Meta

- **漏洞ID**: `CVE-2020-36155`
- **严重程度**: 🔴 CRITICAL
- **风险等级**: 极高风险 (5/5)
- **影响资产**: WordPress 站点
- **预估影响**: 数千万个

**描述**: An issue was discovered in the Ultimate Member plugin before 2.1.12 for WordPress, aka Unauthenticated Privilege Escalation via User Meta. An attacker could supply an array parameter for sensitive metadata, such as the wp_capabilities user meta that defines a user&#x27;s role. During the registration process, submitted registration details were passed to the update_profile function, and any metadata was accepted, e.g., wp_capabilities&#x5B;administrator&#x5D; for Administrator access.


**攻击向量**: 网络扫描

**CVE编号**: CVE-2020-36155

**参考链接**: 
- [参考资料](https://wpscan.com/vulnerability/cf13b0f8-5815-4d27-a276-5eff8985fc0b/)
- [参考资料](https://nvd.nist.gov/vuln/detail/CVE-2020-36155)
- [参考资料](https://wordpress.org/plugins/ultimate-member/#developers)
