### Total Donations Plugin for WordPress &lt; 2.0.6 - Arbitrary Options Update

- **漏洞ID**: `CVE-2019-6703`
- **严重程度**: 🔴 CRITICAL
- **风险等级**: 极高风险 (5/5)
- **影响资产**: WordPress 站点
- **预估影响**: 数千万个

**描述**: Incorrect access control in migla_ajax_functions.php in the Calmar Webmedia Total Donations plugin through 2.0.5 for WordPress allows unauthenticated attackers to update arbitrary WordPress option values, leading to site takeover. These attackers can send requests to wp-admin/admin-ajax.php to call the miglaA_update_me action to change arbitrary options on affected sites. This can be used to enable new user registration and set the default role for new users to Administrator.


**攻击向量**: 网络扫描

**CVE编号**: CVE-2019-6703

**参考链接**: 
- [参考资料](https://wpscan.com/vulnerability/6e6342b0-82ca-4f5f-8b59-92ec3bdf1d02/)
- [参考资料](https://nvd.nist.gov/vuln/detail/CVE-2019-6703)
